Telegram Client
Простой десктопный клиент для Telegram, построенный на библиотеке Telethon и Tkinter. Клиент фокусируется на работе с доверенными контактами, поддерживает уведомления в виде popup-окон, скачивание медиафайлов, системный трей и базовые функции обмена сообщениями. Подходит для использования на Windows (с поддержкой звуковых уведомлений).
Описание
Этот проект представляет собой графический интерфейс для взаимодействия с Telegram API. Он ограничивает список чатов только доверенными контактами (указанными в конфигурации), скачивает медиафайлы в локальную папку и предоставляет удобные уведомления. Клиент работает в фоновом режиме с иконкой в системном трее, минимизируется при закрытии и поддерживает редактирование сообщений.
Проект разделен на модули для удобства разработки:

config: Настройки приложения.
gui: Графический интерфейс (Tkinter).
client: Логика Telegram клиента (Telethon).
utils: Утилиты (работа с медиа, трей).
media: Папка для скачанных файлов.

Возможности
Отображение списка чатов только для доверенных контактов.
Просмотр истории сообщений в выбранном чате.
Отправка текстовых сообщений.
Прикрепление и отправка файлов (фото, видео, документы и т.д.).
Редактирование последнего отправленного сообщения.
Автоматическое скачивание медиафайлов (фото, видео, аудио, голосовые, документы) в папку media/.
Popup-уведомления о новых сообщениях с возможностью быстрого ответа или закрытия.
Мигание имени контакта в списке чатов при новом сообщении.
Звуковые уведомления (на Windows).
Иконка в системном трее для минимизации/восстановления окна.
Автофокус на поле ввода сообщения.
Поддержка 2FA (двухфакторной аутентификации) при авторизации.
Логирование событий для отладки.

Требования
Python 3.8+ (протестировано на 3.12).
Установленные зависимости (см. requirements.txt):
telethon
python-dotenv
pystray
pillow
Для Windows: winsound (встроен в Python).

Установка
Клонируйте репозиторий: git clone https://github.com/EvgeniyKrasnyanskiy/tgwincli.git
cd tgclient
Установите зависимости: pip install -r requirements.txt
Создайте файл .env в корне проекта и заполните его:
API_ID=your_api_id
API_HASH=your_api_hash
PHONE=+your_phone_number
TRUSTED_CONTACTS=contact_id1,contact_id2,contact_id3
Получите API_ID и API_HASH на my.telegram.org.
TRUSTED_CONTACTS: список ID доверенных контактов через запятую (обязательно для ограничения чатов).

Создайте папку media/ (если не существует).

Использование
Запустите приложение: python main.py
При первом запуске:
Клиент запросит код подтверждения (отправляется в Telegram).
Если включена 2FA, введите пароль.

Интерфейс:
Слева: список доверенных контактов (с меткой * для непрочитанных).
Справа: история сообщений и поле ввода.
Кнопки: "Отправить", "📎 Файл" (для прикрепления), "✏️ Ред." (последнее сообщение).
При новом сообщении: popup-уведомление (если окно свернуто) и мигание чата.

Минимизация: Окно сворачивается в трей. Клик по иконке — открыть/свернуть.
Выход: Через меню трея ("Выход").

Конфигурация
.env: Основные настройки (API, телефон, контакты).
config/settings.py: Размеры окон, пути к папкам.
Для изменения списка контактов: обновите .env и перезапустите приложение.

Отладка
Логи записываются в консоль (уровень INFO).
Ошибки: Проверьте .env, интернет-соединение и права доступа к папке media/.

Известные проблемы
Popup-окна могут не работать корректно на некоторых Linux-дистрибуциях (из-за Tkinter).
Нет поддержки групповых чатов (только личные с доверенными ID).
Медиафайлы скачиваются асинхронно, но без индикатора прогресса.
